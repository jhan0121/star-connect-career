# **project: Connecting Star**

## 📌 프로토타입 링크

URL: [링크](https://star-connect-career.lovable.app/)

## 📌 목표

- 커리어에 관한 고민을 가진 사람들 간의 신뢰 기반 커뮤니티 환경을 조성합니다.
- 사용자들이 실질적인 도움을 주고받으며 함께 성장하는 선순환 구조를 만듭니다.

## 📌 문제는 무엇일까?

- 커리어 고민 해결을 위한 정보는 넘쳐나지만, 나에게 꼭 맞는 신뢰할 수 있는 조언을 얻기는 어렵습니다.
- 기존 플랫폼들은 일회성 정보 전달에 그치거나, 멘토와 멘티 간의 신뢰를 보장할 장치가 부족합니다.
- 우리는 **검증된 현직자인 멘토**와 **성장 의지가 있는 사용자인 멘티**를 효과적으로 연결하고, 상호 신뢰를 기반으로 깊이 있는 커리어 상담이 가능한 온/오프라인 플랫폼을 제공하고자 합니다.

## 📌 서비스의 핵심

### 메인 비주얼 및 타이틀

"현직자와의 커피챗, 당신의 커리어 성장을 연결합니다." 와 같은 명확한 타이틀과, 멘토와 멘티가 긍정적으로 소통하는 이미지를 페이지 최상단에 배치하여 서비스의 정체성을 즉각적으로 전달합니다.

### 사용자

- **멘티 (상담 신청자):** 커리어 관련 조언과 상담을 구하며 성장하고자 하는 사용자
- **멘토 (상담 제공자):** 자신의 경험과 지식을 바탕으로 상담을 제공하는 현직 전문가

### 핵심 가치 제안

- **맞춤형 커리어 연결:** 상세하고 직관적인 필터 기능을 통해 나에게 꼭 맞는 현직자 멘토를 찾고, 실질적인 조언을 통해 커리어 로드맵을 설계할 수 있습니다.
- **신뢰 기반 커뮤니티:** 본인 인증, 상호 후기 및 평판 점수 시스템을 통해 멘토와 멘티 모두가 안심하고 참여할 수 있는 안전한 환경을 제공합니다.
- **유연하고 효율적인 소통:** 주기적인 스케줄 설정, 실시간 알림, 1:1 채팅 기능으로 시공간 제약 없이 편리하게 상담을 조율하고 진행할 수 있습니다.

## 📌 사용자 스토리와 인수 조건 (User Stories & Acceptance Criteria)

### 멘티 (상담 신청자)

#### 요구사항: 직관적인 멘토 필터링 기능

- **사용자 스토리:** “멘티로서, 저는 원하는 직무, 경력 등의 조건으로 멘토를 쉽게 필터링하여 찾고 싶습니다. 그래서 저에게 꼭 맞는 멘토를 빠르게 발견할 수 있습니다.”
- **인수 조건:**
  - `Given`: 멘티가 메인 페이지에 접속했을 때
  - `When`: 직무, 경력 등 필터 조건이 기본적으로 노출된다.
  - `And`: 특정 필터 조건을 선택하면
  - `Then`: 해당 조건에 맞는 멘토 리스트가 실시간으로 업데이트되어 표시된다.

#### 요구사항: 멘토 신뢰도 확인 기능

- **사용자 스토리:** “멘티로서, 저는 상담을 신청하기 전에 멘토의 프로필에서 본인 인증 여부와 다른 사람들의 후기, 별점을 확인하고 싶습니다. 그래서 믿을 수 있는 멘토에게 상담을 신청할 수 있습니다.”
- **인수 조건:**
  - `Given`: 멘티가 특정 멘토의 프로필 페이지를 볼 때
  - `When`: 멘토의 이름 옆에 '✔️ 본인 인증 완료' 표시가 있는지 확인한다.
  - `And`: 멘토가 받은 평균 별점과 후기 목록을 조회한다.
  - `Then`: 시스템은 해당 멘토의 인증 정보, 별점, 후기 내용을 정확하게 표시한다.

#### 요구사항: 상담 후 멘토 평가 기능

- **사용자 스토리:** “멘티로서, 저는 상담이 끝난 후 멘토에 대한 별점과 후기를 남기고 싶습니다. 그래서 제 경험을 다른 사용자들과 공유하고, 멘토에게도 피드백을 제공할 수 있습니다.”
- **인수 조건:**
  - `Given`: 멘티가 '내 상담 관리'에서 '완료된 상담'을 선택했을 때
  - `When`: '후기 작성' 버튼을 누르고 별점과 후기 내용을 입력한 후 제출한다.
  - `Then`: 작성된 후기는 해당 멘토의 프로필에 반영되고, '후기 작성' 버튼은 '작성 완료' 상태로 변경된다.

### 멘토 (상담 제공자)

#### 요구사항: 주기적인 상담 가능 시간 설정 기능

- **사용자 스토리:** “멘토로서, 저는 매번 같은 요일과 시간을 입력하는 대신 '매주 금요일 19시'와 같이 주기적인 스케줄을 한 번에 설정하고 싶습니다. 그래서 스케줄 관리를 더 효율적으로 할 수 있습니다.”
- **인수 조건:**
  - `Given`: 멘토가 '상담 가능 시간 관리' 페이지에 접속했을 때
  - `When`: '주기 설정' 옵션을 선택하고 '매주', '금요일', '19:00'를 입력하고 저장한다.
  - `Then`: 시스템은 해당 멘토의 상담 가능 스케줄에 매주 금요일 19시를 자동으로 추가하고, 멘티에게 노출시킨다.

#### 요구사항: 멘티 신뢰도(평판 점수) 확인 기능

- **사용자 스토리:** “멘토로서, 저는 상담 신청을 수락하기 전에 신청한 멘티의 프로필에서 본인 인증 여부와 평판 점수를 확인하고 싶습니다. 그래서 더 안심하고 상담을 진행할 수 있습니다.”
- **인수 조건:**
  - `Given`: 멘토가 '신청받은 상담' 목록에서 특정 멘티의 신청 내역을 볼 때
  - `When`: 멘티의 프로필 정보를 클릭한다.
  - `Then`: 시스템은 해당 멘티의 '✔️ 본인 인증 완료' 여부와 '평판 점수(예: 4.8/5.0)'를 명확하게 표시한다.

#### 요구사항: 상담 후 멘티 평가 기능

- **사용자 스토리:** “멘토로서, 저는 상담이 끝난 후 멘티의 상담 태도에 대해 매너 점수와 간단한 후기를 남기고 싶습니다. 그래서 책임감 있는 커뮤니티 문화를 만드는 데 기여할 수 있습니다.”
- **인수 조건:**
  - `Given`: 멘토가 '완료된 상담' 목록을 볼 때
  - `When`: 특정 상담 건에 대해 '멘티 평가하기' 버튼을 누르고 매너 점수와 코멘트를 입력 후 제출한다.
  - `Then`: 제출된 점수는 해당 멘티의 '평판 점수'에 반영되고, '멘티 평가하기' 버튼은 비활성화된다.

### 공용 (멘티 & 멘토)

#### 요구사항: 본인 인증 기능

- **사용자 스토리:** “사용자로서, 저는 최초 가입 시 본인 인증을 하고 싶습니다. 그래서 플랫폼 내 모든 사용자가 신뢰할 수 있는 환경에서 활동할 수 있습니다.”
- **인수 조건:**
  - `Given`: 사용자가 회원가입을 완료했을 때
  - `When`: 프로필 페이지에 접속한다.
  - `Then`: 시스템은 (프로토타입 상) 본인 인증이 완료되었다고 가정하고, 이름 옆에 '✔️ 본인 인증 완료' 표시를 보여준다.

#### 요구사항: 1:1 채팅 기능

- **사용자 스토리:** “사용자로서, 저는 상담이 확정된 상대방과 간단한 질문이나 시간 조율을 위해 1:1 채팅을 하고 싶습니다. 그래서 더 원활하게 소통할 수 있습니다.”
- **인수 조건:**
  - `Given`: 사용자가 '확정된 상담' 상세 페이지를 볼 때
  - `When`: '채팅하기' 버튼을 누른다.
  - `Then`: 시스템은 상대방과의 1:1 채팅창을 열어주고, 메시지를 주고받을 수 있게 한다.

## 📌 사용자 시나리오 (User Scenario)

### 시나리오 1: 멘티 김취준생

1.  'Connecting Star'에 접속하여 "현직자와의 커피챗, 당신의 커리어 성장을 연결합니다." 라는 명확한 헤드라인을 통해 서비스의 가치를 즉시 파악한다.
2.  메인 화면에 기본으로 펼쳐진 필터에서 **'직무: 마케팅', '상담 주제: 포트폴리오 피드백'**을 선택한다.
3.  실시간으로 필터링된 멘토 리스트에서 별점이 높고 '✔️ 본인 인증 완료' 표시가 있는 A멘토를 선택한다.
4.  A멘토의 프로필에서 "매주 금요일 16시 이후"로 설정된 주기적인 상담 시간을 확인하고 원하는 시간을 선택, 자신의 포트폴리오 파일을 첨부하며 상담을 신청한다.
5.  잠시 후, 'A멘토가 상담을 수락했습니다' 라는 푸시 알림을 받는다.
6.  상담 완료 후, A멘토에게 별점 5점과 함께 감사 후기를 남기고 다른 사용자들을 위해 '추천하기' 버튼을 누른다.

### 시나리오 2: 멘토 최시니어

1.  '내 정보' 페이지에서 상담 가능 시간을 "매주 수요일 19시 이후"로 주기성 설정을 완료해 둔다.
2.  새로운 상담 요청 알림을 받고 앱을 확인한다.
3.  신청자 B의 프로필에서 '✔️ 본인 인증 완료' 표시와 '평판 점수: 4.8/5.0' 을 확인하고 안심하며 '수락' 버튼을 누른다.
4.  다른 신청자 C는 자신의 전문 분야와 거리가 있어 '거절' 버튼을 누르고, "제 전문 분야와 달라 다른 멘토님께 더 좋은 도움을 받으실 수 있을 것 같습니다." 라는 거절 사유를 선택하여 보낸다.
5.  '내 상담 관리' 페이지의 캘린더 뷰를 통해 확정된 상담 일정을 한눈에 파악한다.
6.  B멘티와 상담 완료 후, B의 성실한 태도에 대해 매너 점수 5점과 긍정적인 코멘트를 남겨 B의 평판 점수에 기여한다.

## 📌 핵심 기능 (Core Features)

1.  **직관적인 탐색 및 신뢰도 확인**
    - **메인 이미지 & 타이틀:** 서비스의 가치를 명확히 전달하는 첫인상 제공.
    - **통합 필터 시스템:** 사용자가 원하는 조건(직무, 경력, 주제 등)의 멘토를 클릭 몇 번만으로 쉽게 찾을 수 있도록 필터를 기본으로 노출.
    - **신뢰도 프로필:** 모든 사용자의 프로필에 '✔️ 본인 인증' 여부를 표시하고, 멘티에게는 '**평판 점수**'를, 멘토에게는 '**상담 후기 및 별점**'을 노출하여 상호 신뢰를 높임.
2.  **효율적이고 유연한 상담 관리**
    - **주기적인 스케줄 설정:** 멘토가 "매주 금요일" 등 반복되는 상담 가능 시간을 쉽게 설정하여 관리 부담을 줄임.
    - **상담 관리 대시보드:** '내 상담' 페이지에서 '신청한 상담'과 '신청받은 상담'을 탭으로 분리하고, 캘린더 뷰를 제공하여 일정을 직관적으로 관리.
    - **상호 피드백 시스템:** 상담 종료 후 멘토는 멘티에게 매너 점수와 후기를, 멘티는 멘토에게 별점과 후기, 추천을 남겨 건전한 커뮤니티 문화를 조성.
3.  **원활한 커뮤니케이션 지원**
    - **실시간 알림:** 상담 신청, 수락, 거절, 메시지 수신 등 주요 상황을 놓치지 않도록 즉시 알림 전송.
    - **1:1 채팅:** 상담 확정 전후, 간단한 질문이나 세부 일정 조율을 위해 사용자들이 1:1로 대화할 수 있는 채널 제공.
    - **거절 사유 전달:** 멘토가 상담을 거절할 경우, 미리 정의된 사유를 선택하거나 직접 입력하여 멘티에게 전달함으로써 긍정적인 사용자 경험을 유지.
